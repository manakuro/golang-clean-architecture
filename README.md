# golang-clean-architecture
Go with Clean Architecture

## DeepWiki
https://deepwiki.com/manakuro/golang-clean-architecture

## Run Docker

```
$ cd docker
$ docker compose up
```

## Installation

```
$ make install
```

## Set up database

```
$ make setup_db
```

## Migrate schema

```
$ make migrate_schema_up
```

## Seed data

```
$ make seed
```

## Start server

```
$ make start
```
